# Cafe Management System

## 📌 Overview

The Cafe Management System is a web-based application developed using PHP and MySQL to manage cafe operations such as user authentication and customer interaction.

## 🚀 Features

* User Signup & Login System
* Session Management (Login/Logout)
* Database Integration
* Responsive UI Design

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: PHP
* Database: MySQL

## ⚙️ How to Run

1. Install XAMPP
2. Move project folder to `htdocs`
3. Import `final.sql` into phpMyAdmin
4. Open `localhost/Cafe-Management-System/public`
